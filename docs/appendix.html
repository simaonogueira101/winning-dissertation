7 Appendix

7.1 Crunchbase Enterprise API Documentation

The following tables include information from the documentation of Crunchbase’s Enterprise API (Crunchbase 2021a) and describe the structure of this same interface.

 

Table 7.1: Crunchbase: Location Data.
Variable Name Description
location Full location name (e.g. Denver, Colorado, United States, North America)
location_identifiers Globally unique id of this entity
Request URL: https://api.crunchbase.com/api/v4/searches/locations/
Table 7.2: Crunchbase: Accelerator Data.
Variable Name Description
identifier Name of the Organization
description Organization Description
category_groups Superset of Industries (e.g. Software, Mobile, Health Care)
demo_days Whether an accelerator hosts any demo days
founder_identifiers Founders of the organization
investor_type This describes the type of investor this organization is (e.g. Angel, Fund of Funds, Venture Capital)
location_identifiers Where the organization is headquartered
program_duration The duration of the Acceleration Program in number of weeks
program_type The type of Accelerator Program (e.g. On-Site, Online)
website Link to homepage
Request URL: https://api.crunchbase.com/api/v4/searches/organizations/
Table 7.3: Crunchbase: Accelerator Founder Data.
Variable Name Description
identifier First and last name of a Person
num_founded_organizations Number of Organizations that the person founded
num_investments Number of Investments the Individual has participated in
num_jobs Number of jobs the individual has had
degrees The academic degrees that the person has received
Request URL: https://api.crunchbase.com/api/v4/entities/people/
Table 7.4: Crunchbase: Accelerator Investment Data.
Variable Name Description
identifier First and last name of a Person
num_founded_organizations Number of Organizations that the person founded
num_investments Number of Investments the Individual has participated in
num_jobs Number of jobs the individual has had
degrees The academic degrees that the person has received
Request URL: https://api.crunchbase.com/api/v4/searches/investments/
Table 7.5: Crunchbase: Startup Data.
Variable Name Description
identifier Name of the Organization
description Organization Description
funding_stage This field describes an organization’s most recent funding status (e.g. Early Stage Venture, Late Stage Venture, M&A)
funding_total Total amount raised across all funding rounds
location_identifiers Where the organization is headquartered
valuation Latest post-money valuation of the organization
raised_investments All investments received by the organization
ipos The organization’s initial public offering
Request URL: https://api.crunchbase.com/api/v4/entities/organizations/
